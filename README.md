 ⌚ Smartwatch Interface

Um projeto de interface minimalista e moderna de um Smartwatch, desenvolvido para praticar a integração entre estrutura HTML, estilização avançada com CSS (incluindo animações) e lógica de programação em JavaScript para manipulação do DOM em tempo real.

---

## 🛠️ Tecnologias Utilizadas

O projeto foi construído do zero utilizando a tríade fundamental do Front-End:

*   **HTML5:** 
*   **CSS3:** Estilização baseada em design minimalista, utilizando **Flexbox** para alinhamentos,  `@keyframes` para criar um equalizador de música animado.
*   **JavaScript:** Criação do motor do relógio utilizando o objeto nativo `Date()`, manipulação dinâmica de elementos via `querySelector` e atualização em tempo real de 1 em 1 segundo com `setInterval`.

---

## 💡 Funcionalidades

*   **Relógio Digital Dinâmico:** Sincroniza com o horário real do sistema do usuário (horas e minutos).
*   **Equalizador de Áudio Animado:** Barras de áudio que pulsam em tempos diferentes utilizando seletores `:nth-child` e `animation-delay` no CSS.
*   **Layout Responsivo e Clean:** Cores modernas, contraste acessível e design inspirado em interfaces premium de smartwatches atuais.
